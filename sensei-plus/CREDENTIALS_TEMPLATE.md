# 本運用用 - 教員パスワード管理表

**⚠️ 注意：このファイルは絶対に GitHub に含めないでください**

## 🔐 パスワード生成手順

### **Step 1: パスワード生成ツールを実行**

以下の URL を開く（ローカル開発環境で）：
```
tools/password-generator.html
```

または、ブラウザのコンソール（F12）で以下を実行：
```javascript
// password-generator.js を読み込んだ後
generateAllCredentials();
```

### **Step 2: パスワードを保存**

生成されたパスワード一覧がコンソールに表示されます：

```
┌─────────┬──────────────────────────┐
│ (index) │ Values                   │
├─────────┼──────────────────────────┤
│ test    │ 'aBcD1234!@#$%^&*'      │
│ T001    │ 'xYz9876#$%^&*!@'       │
│ T002    │ 'pQr5432@#$%^&*!a'      │
└─────────┴──────────────────────────┘
```

### **Step 3: auth.js に記入**

以下のテキストをコピーして、`homepage/auth.js` に貼り付け：

```javascript
const VALID_TEACHERS = {
    'test': 'aBcD1234!@#$%^&*',
    'T001': 'xYz9876#$%^&*!@',
    'T002': 'pQr5432@#$%^&*!a'
};
```

## 🔒 パスワード配布方法（推奨）

### **方法 1: セキュアリンク（推奨）**

```
こちらのセキュアリンクからパスワードを取得してください：
https://[セキュアサイト]/teacher/T001
```

### **方法 2: 初期セットアップ画面**

アプリ初回起動時にパスワード変更を強制：
```javascript
if (firstLogin) {
    showPasswordChangeScreen();
}
```

### **方法 3: QR コード配布**

```
QR コード化されたパスワード + ID
（スマホで読み込むと自動ログイン）
```

## ⚠️ セキュリティチェックリスト

- [ ] パスワードは 12 文字以上
- [ ] パスワードに大文字・小文字・数字・記号を含む
- [ ] 教員ごとにユニークなパスワード
- [ ] パスワードを平文で送信しない
- [ ] GitHub には含めない
- [ ] ローカルで保管

---

## 🔄 パスワード変更手順

1. 新しいパスワードを生成
2. auth.js を更新
3. GitHub にプッシュ（.gitignore に secrets.js を追加）
4. 教員に新しいパスワードを配布

---

## 📞 緊急時の対応

**パスワードが漏洩した場合：**
1. 新しいパスワードを即座に生成
2. auth.js を更新し GitHub にプッシュ
3. 全教員に通知
4. 前のパスワードは使用不可に
